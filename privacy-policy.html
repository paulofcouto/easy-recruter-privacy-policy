Política de Privacidade

Esta extensão do Chrome, "Salvar URL", respeita a sua privacidade. Não coletamos, armazenamos ou compartilhamos dados pessoais sem o seu consentimento explícito. Os dados capturados pela extensão são utilizados exclusivamente para a funcionalidade pretendida: envio de informações de perfil do LinkedIn para sua API interna.

Finalidade do Tratamento de Dados

Os dados coletados, como nome, contato e informações profissionais, são utilizados exclusivamente para manter um banco de talentos. A finalidade principal é permitir que possamos entrar em contato com candidatos sobre oportunidades de trabalho que possam surgir e sejam compatíveis com seu perfil.

Base Legal para o Tratamento

De acordo com a Lei Geral de Proteção de Dados (LGPD), utilizamos os dados coletados com base no princípio do "legítimo interesse", conforme o Art. 7º, inciso IX. Isso significa que os dados são tratados para propósitos que sejam necessários e proporcionais, sempre com o objetivo de facilitar a comunicação com os candidatos sobre oportunidades de emprego relevantes.

Direitos dos Titulares de Dados

Em conformidade com a LGPD, os titulares têm o direito de:

  • Solicitar a exclusão de seus dados do banco de talentos;
  • Requerer informações sobre como seus dados estão sendo utilizados;
  • Atualizar ou corrigir informações pessoais;
  • Revogar o consentimento para o uso de dados, quando aplicável.

Para exercer esses direitos, entre em contato através do e-mail: paulofernandescouto@gmail.com.

Período de Retenção

Os dados permanecerão armazenados em nosso sistema pelo período necessário para cumprir as finalidades mencionadas ou até que o titular solicite sua exclusão.

Dados Sensíveis

Esta extensão não coleta nem trata dados sensíveis, como definidos pela LGPD: informações sobre origem racial ou étnica, convicção religiosa, opinião política, filiação sindical, dados de saúde, genéticos, biométricos, vida sexual ou orientação sexual. Caso esses dados sejam coletados em cenários futuros, o consentimento explícito será solicitado.
